# cn-home-guide

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面集合的托管与分发点。

## 项目简介

本仓库旨在整理和发布一系列独立的 HTML 页面。这些页面可能涵盖信息展示、工具页面、说明文档等多种用途，彼此之间无强关联，各自独立运行。仓库本身不提供任何导航、搜索或聚合功能，仅作为静态页面文件的存储与访问入口。

## 目录说明

```
.
├── index.html          # 仓库根页面（可选，默认展示）
├── pages/              # 存放所有独立 HTML 页面
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── README.md           # 本文件
```

所有页面文件均存放于 `pages/` 目录下，文件名与页面内容相对应。根目录下的 `index.html` 可能用于展示仓库基本信息或页面列表，但不是必须的。

## 页面归档说明

- 每个 HTML 文件均为独立、自包含的页面，不依赖本仓库其他文件。
- 页面内容、样式、脚本均内嵌于单个文件中，便于直接通过浏览器打开或部署。
- 归档的页面可能来自不同时期、不同用途，内容与风格可能不一致。
- 本仓库不对页面内容的准确性、时效性或完整性做任何保证。

## 维护说明

- 本仓库由维护者根据个人需求不定期更新。
- 欢迎通过 Issue 提交建议或报告问题，但请注意本仓库不针对任何具体网站或服务。
- Pull Request 需确保新增页面为独立、自包含的 HTML 文件，且不包含任何外部依赖。
- 维护者保留对页面内容进行修改或删除的权利。

## 使用方式

你可以直接通过浏览器访问本仓库的 GitHub Pages 或直接下载页面文件到本地使用。每个页面均可独立运行，无需额外配置。
